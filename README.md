# 📊 Power BI Sales Dashboard

An interactive Power BI dashboard designed to analyze sales performance, shipment trends, product insights, and profitability metrics. This report provides a clear overview of business performance using dynamic visuals and filters.

---

## 🚀 Project Overview

This dashboard helps stakeholders monitor:

* Total Sales & Revenue
* Shipment Volume
* Profit & Profit Percentage
* Product-wise Performance
* Salesperson Performance
* Geographic Distribution

The report enables quick decision-making through visual insights.

---

## 📌 Key Metrics

* **Total Amount:** $141M
* **Total Boxes:** 9M
* **Shipment Count:** 25K
* **Total Profit:** $81M
* **Profit %:** 57.3%

---

## 📊 Dashboard Features

### 1. Sales Performance Analysis

* Year-over-Year comparison (CY vs PY)
* Tracks growth trends in revenue and boxes

### 2. Shipment Distribution

* Visual representation of shipment frequency
* Identifies peak shipment ranges

### 3. Top Products Analysis

* Displays best-performing products
* Highlights revenue contribution by each product

### 4. Salesperson Performance

* Ranking of top 6 salespersons
* Metrics include:

  * Total Amount
  * Total Boxes
  * Profit %

### 5. Geographic Insights

* Sales distribution by region/country
* Helps identify strong and weak markets
